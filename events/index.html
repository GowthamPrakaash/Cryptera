BLIND CODING

Do you have complete faith in your brain and hand? Can your hands produce what your brain thinks? Well may be this is the event you have been waiting for. A coding event with a twist Blind Coding is something which will bring out your power of concentration and will.

Rules:

  • Each participant has to write the code for the problem statement,with the MONITOR OFF.
  • Participants whose code gets executed will be given higher preference
  • The second preference will be given to code with least number of errors.
  • In case of tie in number of errors then timing will be considered.
  • Shortlisted participants of Round 1 will appear for Round 2 for which above rules will be applicable.

Round 1:

  • A simple code with syntax error will be given on paper
  • Participants have to correct the errors on paper and type the code with MONITOR SWITCHED OFF.
  • Participant should then work on it within given time of 10 minutes.
  • Top 10 teams with highest score would be short listed

Round 2:

  • Only problem statement will be provided
  • Participants need to type the code in TURBO C with MONITORS SWITCHED OFF
  • 20 minutes will be given to each participant.
  • Top 5 teams with highest score would be short listed to Finals.

CODE HUNT

An input and output will be given and the programmer should find the code to bring out that output from the given input. Here is chance for you to become an efficient programmer! This event is all about to hunt the code to produce the result as described. And also the code should be an efficient one…!

Rules:

  • Maximum 2-3 members per team.
  • Participants should not use mobile phones.
  • No access to the Internet.

Round 1:

  • An input and an output will be given.
  • Participants should find the relevant code within the given time.
  • Top 10 teams with highest score would be short listed to second round.

Round 2:

  • A theoretical problem will be given and the participants should find an efficient code to solve the problem within the given time.

FINALS

  • Round 2 would be considered as final round.
  • Top 2 teams with highest score would be awarded as winners.
